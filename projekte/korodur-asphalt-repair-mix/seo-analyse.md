## 2. On-Page SEO-Anforderungen

### 2.1 Title Tag

**Empfehlung (max. 60 Zeichen):**

```
Asphaltreparatur dauerhaft in 30 Min. | Rapid Set ARM
```

**Zeichenzahl:** 55

**Alternativ-Varianten:**
- `Rapid Set ARM | Dauerhafte Asphaltreparatur ohne Fräse` (54 Zeichen)
- `Schlagloch dauerhaft reparieren | Rapid Set ARM` (50 Zeichen)
- `Asphaltreparatur in 30 Min. | KORODUR Rapid Set` (50 Zeichen)

**Logik:** Der Title Tag enthaelt das Hauptkeyword ("Asphaltreparatur"), den zeitlichen USP ("30 Min.") und den Produktnamen ("Rapid Set ARM"). Die Marke KORODUR kann im Title entfallen, da sie in der Domain (korodur.de) bereits sichtbar ist -- Google zeigt die Domain automatisch mit an.
